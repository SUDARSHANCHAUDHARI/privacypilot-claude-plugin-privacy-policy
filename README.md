# PrivacyPilot — Privacy Policy

This repository hosts the privacy policy for the **PrivacyPilot** Claude Code plugin.

**Live URL:** https://sudarshanchaudhari.github.io/privacypilot-claude-plugin-privacy-policy/

---

PrivacyPilot is a Claude Code plugin that automatically generates Play Store-compliant privacy policies for Android apps by scanning `AndroidManifest.xml` and Gradle dependencies.

**Plugin repository:** https://github.com/SUDARSHANCHAUDHARI/PrivacyPilot
