Privacy Policy

Last updated: April 6, 2026

This Privacy Policy applies to the PrivacyPilot Claude Code plugin, developed by SUDARSHANCHAUDHARI / SudarshanTechLabs.

What PrivacyPilot Does

PrivacyPilot is a Claude Code plugin that reads your local Android project files (AndroidManifest.xml, build.gradle) and generates a privacy policy HTML file on your machine.

Data Collection

PrivacyPilot does not collect, transmit, or store any user data. Specifically:

  • All file reading and policy generation happens locally on your machine
  • No data is sent to any external server by this plugin
  • Your developer details (name, email, GitHub username) are stored locally in your Claude Code settings on your machine only
  • No analytics, crash reporting, or telemetry is included in this plugin

Third-Party Services

PrivacyPilot itself uses no third-party services. The generated privacy policies it produces may reference third-party services (AdMob, Firebase, etc.) on behalf of your Android app — those are disclosures about your app, not about this plugin.

GitHub Pages Deployment

When you use the /privacypilot:github-page command, the plugin uses your local git configuration to push an HTML file to a GitHub repository you own. No data passes through any PrivacyPilot servers.

Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected by updating the "last updated" date above.
